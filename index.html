Healthy Food and Recipe Applet

Start creating your own healthier meals!

This applet allows simple searches of the USDA Food Composition Databases and will return a food item which closely matches the user input. The data returned is the USDA standard reference name (w/UPC) for brand name foods as well as the calories, fat, sugars and carbohydrate content.

Once you've found the food you are looking for, use the video search component to retrieve recipes that include your food. The search will return the 10 YouTube videos matching user input.
